# Qt-SmartHome
嵌入式智能家居系统

使用qt界面客户端并且自定义服务端连接数据库，通过http访问服务端数据和外部api,qt客户端直连mqtt broker和硬件进行数据交换，硬件由stm32f103c8t6作开发板，esp8266做wifi模块连接mqtt broker
